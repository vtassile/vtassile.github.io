Aplicaciones Web
Somos Facta

Sistema Multiusuario y multinivel de Gestión de Comedor Universitario. Desarrollado en Angular y Express. Permite consultas, pago retivo de Viandas de Comedor. Los movimientos financieros de cada comensal, la carga y control de menús diarios por tipo de comida y lugar de consumo.

Frutos Visión

Aplicación Web que permite la Detección de Frutos a través de YOLO, un algoritmo de visión artificial basado en redes neuronales convolucionales. Una aplicación en el Alto Valle de Río Negro.

Pro Williams

Aplicación Web que demuestra el uso de Modelos Mixtos No Lineales para el Pronóstico de cosecha para frutos de pera correspondientes al cultivar William´s para el Alto Valle de los rios Neuquén, Limay y Negro en la Patagonia Argentina.

Desnutrición

Aplicación Web que permite la Modelación y Presentación Visual de Indicadores de Malnutrición Infantil en contextos de Vulnerabilidad Social y Ambiental. Se describen 50.000 registros antropométricos asociados a datos del sistema de salud.
